Deploy
1. Create data base in sql explorer.
2. Publish Book Library Data Base project to created data base.
2. Update Book_Library_EF_Core_Proxy_Class_Library.Constants.LibraryConstants.CONNECTIONSTRING 
through created data base Connection String in Book Library EF Core Proxy Class Library project. 
3. Rebuild solution.
4. Run in Visual Studio Book Library .NET Core WPF App project or run exe file in bin directory.
